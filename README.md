## Overview
This project presents an interactive **E-commerce Sales Dashboard** built using **Power BI**. The dashboard converts raw sales data into visual insights to support decision-making across category performance, revenue trends, and customer behavior.

## Key Visualizations
- **Category Performance:** Bar/column charts highlighting profit and quantity distribution.
- **Trend Analysis:** Line charts showing monthly and quarterly sales fluctuations.
- **Payment Mode Insights:** Pie/Donut charts showing payment preference patterns.
- **KPI Cards:** Total sales, total profit, quantity sold, and margins.
- **Interactive Filters:** Drill-downs and slicers for dynamic exploration.

## Power BI Features Used
- Power Query for data cleaning
- Data modeling with relationships
- DAX measures for KPIs
- Custom and native Power BI visuals
- Slicers, tooltips, and drill-down features

## Objective
Provide business users with a clear, interactive view of sales performance to support:
- Inventory decisions
- Category optimization
- Operational planning
- Forecasting and revenue strategy

## How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Use slicers to filter by category, payment mode, region, or period.
3. Hover on charts for tooltips.
4. Explore drill-downs for deeper insights.
